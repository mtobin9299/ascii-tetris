Module Input

module Input: sig .. end

The abstract input type

type t 
type key_input =
| KMoveRight
| KMoveLeft
| KHold
| KRotateRight
| KRotateLeft
| KHardDrop
| KQuit
| KRestart

key_input is the type that represents the possible key to action bindings

val init : unit -> t

init () is the intitial input state with key map derived from a json file

val process_input : t -> string -> Game_state.t -> Game_state.t

process_input input_state ch game_state is the new game state following the execution of the command binded to ch in input_state

val has_quit : t -> bool

has_quit input_state is true if the player has pressed the quit key. False otherwise.

val get_key : key_input -> t -> char

get_key key_input input_state is the char binded to key_input in input_state
