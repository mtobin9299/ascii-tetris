Module Tetromino

module Tetromino: sig .. end

orientation is the type representing the four standard orientations a tetromino can assume

type orientation =
| Zero
| Right
| Two
| Left 
type piece_type =
| I
| J
| L
| T
| S
| Z
| O

piece_type is the type representing the seven different pieces there are in tetris

type pos = int * int

pos is the type that represents an x, y coord pair

type t 
val init_tetromino : pos -> piece_type -> t

init_tetromino pos piece_type is a new tetromino with position pos of type piece_type

val piece_type : t -> piece_type

piece_type tetromino is the piece type of tetromino

val orientation : t -> orientation

orientation tetromino is the orientation of tetromino

val offsets : t -> (orientation * pos list) list

offsets tetromino is the standard list of all rotation offsets for each possible orientation of tetromino

val initial_positions : t -> pos list

initial_positions tetromino is the list of the relative, initial positions of the minos that make up tetromino

val pos : t -> pos

pos tetromino is the world position of tetromino

val minos_world_positions : t -> pos list

minos_world_positions tetromino is the list of world positions of the minos that make up tetromino

val rotate_minos : orientation -> pos list -> pos list

rotate_minos orientation minos is the list of minos with relative positions of minos in the orientation of orientation

val change_orientation : orientation -> t -> t

change_orientation orientation tetromino is tetromino with new orientation orientation

val change_pos : pos -> t -> t

change_pos pos tetromino is tetromino with new position pos

val move : pos -> t -> t

move delta_pos tetromino is tetromino with delta_pos added to its pos
